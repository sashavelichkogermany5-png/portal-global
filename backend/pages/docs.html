Master overview

Use this page to understand the current release runtime, the main public routes, and the deployment posture of PORTAL Global.

Quick start

  1. Open /register or /login.
  2. Enter the main workspace at /app.
  3. Use /orders when you need a real order record.
  4. Use /help and /pricing for rollout support and plan context.

Release runtime

Primary runtime

The current release candidate is the unified server.js runtime on one main public port behind a reverse proxy.

  • / (landing)
  • /app (workspace)
  • /orders
  • /pricing
  • /master-overview
  • /help

API and data

Public pages and authenticated API requests stay on the same origin under /api/*. The default database is SQLite.

Primary runtime: server.js on port 3000 API: same origin under /api/* Database (default): database/portal.db

Key runtime endpoints

POST /api/auth/register POST /api/auth/login POST /api/auth/logout GET /api/auth/me GET /api/health GET /api/stats GET /api/orders POST /api/orders GET /api/usage POST /api/feedback POST /api/billing/create-checkout-session

Operator-only and environment-specific endpoints are intentionally omitted from the public overview.

Production checklist

  • Set NODE_ENV=production and configure PORT=3000.
  • Use HTTPS end-to-end; secure cookies require TLS.
  • Set APP_ORIGIN and ALLOWED_ORIGINS to the public domain.
  • Review rate limits via RATE_LIMIT_WINDOW_MS and RATE_LIMIT_MAX.
  • Store secrets in .env.production or host environment variables.

HTTPS requirement

Sessions use secure cookies in production, so HTTPS is required for login and API access.

Reverse proxy example (nginx)

server { listen 443 ssl; server_name portal.example.com; location / { proxy_pass http://127.0.0.1:3000; proxy_set_header Host $host; proxy_set_header X-Forwarded-Proto $scheme; proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for; } }

Operator-only surfaces

Administrative and control routes remain access-controlled. The main public product flow should stay on landing, auth, workspace, pricing, help, and the master overview.
